# RichTextBox Trial

ASP.NET Core rich text editor with 100 server-side demos.

## Quick Start

```
dotnet run --project RichTextBox.WebSite
```

Open the URL printed in the console and browse `/Demos`.

## What's Included

- **lib/RichTextBox.dll** - Editor library
- **RichTextBox.WebSite/** - Demo website with 100 working examples
- **RichTextBox.lic** - Trial license file (30-day evaluation)
- **docs/** - Setup and deployment guides

## Requirements

- .NET 8 SDK
